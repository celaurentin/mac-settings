Instant show/hide Dock:

Keyboard -> Shortcuts -> App Shortcuts -> Add the following mappings to CMD-ESC: <br />
Toggle Full Screen  <br />
Fullscreen          <br />
Enter Full Screen   <br />
Exit Full Screen    <br />
